# Deployment Security Checklist

Use this checklist before every deployment to ensure security best practices.

## Pre-Deployment

### Code Security
- [ ] Run `npm run security:audit` - 0 vulnerabilities
- [ ] No hardcoded secrets in code (check all `.ts` files)
- [ ] Environment files contain only non-sensitive values
- [ ] All `bypassSecurityTrust*` uses are documented with security comments
- [ ] No use of `innerHTML`, `eval`, or `document.write` with user data

### Dependencies
- [ ] `package-lock.json` is committed and up-to-date
- [ ] All dependencies are from trusted sources (official Angular, well-known packages)
- [ ] No deprecated packages in use
- [ ] License compatibility verified

### Build
- [ ] Production build succeeds: `npm run build`
- [ ] No console warnings in build output
- [ ] Bundle size is reasonable (<2MB total)
- [ ] Source maps are NOT included in production build

## Deployment Configuration

### Netlify/Hosting
- [ ] `netlify.toml` includes all security headers
- [ ] HTTPS enforcement configured (HTTP → HTTPS redirect)
- [ ] Custom domain SSL certificate is valid
- [ ] Deploy previews are protected (if needed)

### Security Headers Verification
After deployment, verify at https://securityheaders.com/:
- [ ] Content-Security-Policy present
- [ ] X-Frame-Options: DENY
- [ ] X-Content-Type-Options: nosniff
- [ ] Referrer-Policy configured
- [ ] Permissions-Policy configured

### SSL/TLS Configuration
Test at https://www.ssllabs.com/ssltest/:
- [ ] Grade A or A+ rating
- [ ] Certificate is valid and trusted
- [ ] TLS 1.2+ only (no TLS 1.0/1.1)
- [ ] Strong cipher suites

## Post-Deployment

### Functional Testing
- [ ] Homepage loads correctly
- [ ] All navigation links work
- [ ] Contact form submits successfully
- [ ] Career form submits successfully
- [ ] Google Maps embed displays correctly
- [ ] No console errors in browser

### Security Scanning
- [ ] Mozilla Observatory scan: https://observatory.mozilla.org/
  - Target grade: A or A+
- [ ] SecurityHeaders.com scan shows all headers
- [ ] Sucuri SiteCheck: https://sitecheck.sucuri.net/
  - No malware detected
  - Not blacklisted

### Performance & Accessibility
- [ ] Lighthouse score > 90 for Performance
- [ ] Lighthouse score > 90 for Accessibility
- [ ] Lighthouse score 100 for Best Practices
- [ ] Page load time < 3 seconds

## Access Control

### Repository
- [ ] Repository is private (or access controlled)
- [ ] 2FA enabled for all team members
- [ ] Branch protection rules configured
- [ ] No force-push to main branch

### Hosting Platform
- [ ] 2FA enabled on Netlify/hosting account
- [ ] Deploy keys rotated if compromised
- [ ] Environment variables properly configured
- [ ] Build hooks secured with tokens

## Monitoring & Maintenance

### Regular Tasks
- [ ] Weekly: Run `npm run security:audit`
- [ ] Monthly: Update dependencies (`npm update`)
- [ ] Monthly: Security header scan
- [ ] Quarterly: Full security audit
- [ ] Quarterly: SSL certificate expiry check

### Incident Response
- [ ] Security contact email configured
- [ ] Backup/restore procedure documented
- [ ] Rollback procedure tested
- [ ] Team notified of deployment
